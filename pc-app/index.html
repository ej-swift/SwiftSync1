Put multichat inside OBS

Same URL for every streamer. It is not a personal link — 127.0.0.1 means this PC only. OBS loads chat from SwiftSync running on the same computer. No pairing code needed. If your relay uses another port, the URL below updates automatically.

① SwiftSync PC app open ② Chat connected (Connect above)

  1. Open OBS Studio on this PC (same machine as SwiftSync).
  2. Menu: View → Docks → Custom Browser Docks…
  3. Dock name: SwiftSync Chat (any name is fine).
  4. Paste this URL exactly, then click Apply:
http://127.0.0.1:4000/dock/chat.html

Right-click Copy URL to copy all steps for Discord or notes.

The dock appears as a panel inside OBS (resize by dragging edges). Use Refresh cache of current page in the dock’s ⋮ menu if chat stays on “Connecting…”.
